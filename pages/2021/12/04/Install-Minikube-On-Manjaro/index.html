Manjaro 搭建 minikube 环境

by niuhe
in Kubernetes on 2021-12-04 | tags: k8s | comments

安装 minikube

  1. 安装依赖
sudo pacman -Sy libvirt qemu ebtables dnsmasq

sudo usermod -a -G libvirt $(whoami)
newgrp libvirt

sudo systemctl start libvirtd.service
sudo systemctl enable libvirtd.service

sudo systemctl start virtlogd.service
sudo systemctl enable virtlogd.service
  1. 装docker和kvm2 驱动
sudo pacman -Sy docker-machine

yaourt -Sy docker-machine-driver-kvm2
  1. 装minikube
yay -Sy minikube kubectl-bin

minikube version
whereis kubectl
kubectl -h
  1. 装环境

注意关掉 vpn

minikube start --vm-driver kvm2 --registry-mirror=https://dockerhub.azk8s.cn --image-repository=registry.cn-hangzhou.aliyuncs.com/google_containers


minikube status
kubectl cluster-info
kubectl get nodes
